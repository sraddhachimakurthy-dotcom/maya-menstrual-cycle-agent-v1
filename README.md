# Maya 🌸

Maya is an AI-powered menstrual health companion.

## Features

- Symptom tracking
- Pain severity logging
- Period prediction
- Personalized insights
- Emotional awareness
- Conversation memory
- AI responses using Gemini

## Technologies

- Python
- Google Gemini API
- SQLite
- Pandas
- Google Colab

## Future Improvements

- Web application
- Mobile app
- Dataset intelligence
- User authentication
